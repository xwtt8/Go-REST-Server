This project is the backend Server for 'Rice Now'.
'Rice Now' is an information pushing application on android system that can provide students with real time info about their courses information, class online discussion, instructors' post and many more usefule information so that people do not need to look for them explictly

This backend server is mainly written in golang by using REST API to host content and communication with front end devices. It also use mongoDB for stroing data that are gathered by our web crawler
